# Install the Everyday PA System — 3 steps, no tech background needed

## 1 · Install the plugin file
You received a file ending in **`.plugin`** (e.g. `everyday-pa-system-v0.8.1.plugin`).
- Open the **Claude desktop app** → **Settings → Plugins** (under "Customize") → choose
  **Install plugin** / **Add plugin** and pick the file.
- If your version shows no such button: double-click the `.plugin` file itself and let the
  Claude app open it.
- **You'll know it worked when:** "Everyday PA System" appears in your Plugins list. (Its page
  may show a skill named "/pa-system" — ignore it; plain phrases do everything.)

## 2 · Open Claude COWORK — not a plain chat
Setup needs Cowork (it can create your folder on your computer and show progress).
In the app, start a **new Cowork session** — look for the **Chat / Cowork** switch and make
sure **Cowork** is selected.

## 3 · Say the magic phrase
Type exactly: **Set up my Everyday PA System**
The wizard takes it from there — it welcomes you, creates your folder, meets you, proves the
system survives sessions, and lands your first win. About 10 minutes. You only answer, approve,
and confirm.

---
*Interrupted mid-setup? Open a new Cowork chat and say **"Set up my Everyday PA System"** again —
it resumes exactly where you stopped. **After setup, daily use is even simpler: open a Cowork
chat inside your "Everyday PA System" Project and just say hello.** Something wrong later? A `HELP.md`
file lives in your "Everyday PA System" folder with every fix and prompt you might need.*
